💡 Elemen Deprecated Dasar

Berikut adalah contoh elemen HTML yang sudah tidak direkomendasikan lagi digunakan.

deprecated.html
<!-- Contoh elemen deprecated -->
<center>Teks di tengah</center>
<font color="red">Teks Merah</font>
<big>Teks Besar</big>
<marquee>Teks Bergerak</marquee>

🔍 Penjelasan Detail:

• <center> - Gunakan CSS text-align: center;
• <font> - Gunakan CSS color, font-size
• <big> - Gunakan CSS font-size
• <marquee> - Gunakan animasi CSS

⚡ Pro Tips

Hindari elemen ini pada proyek modern karena tidak kompatibel dengan HTML5.
