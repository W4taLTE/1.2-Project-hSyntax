üí° Optimasi Performa

Elemen teknis untuk meningkatkan kecepatan dan efisiensi halaman web.

‚Ä¢ preconnect - Membuat koneksi awal ke domain eksternal
‚Ä¢ dns-prefetch - Resolve DNS lebih awal untuk domain eksternal
‚Ä¢ Critical CSS - CSS penting dalam <style> untuk render cepat
‚Ä¢ Teknik loadCSS - Memuat stylesheet non-blocking dengan media print

‚ö° Pro Tips
