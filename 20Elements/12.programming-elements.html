📝Basic Tables

💡 Info Boxes

Table digunakan untuk menampilkan data dalam format baris dan kolom. Elemen utama adalah <table>, <tr>, <th>, dan <td>.

table.html
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
</table>

🔍 Penjelasan Detail:

• <table> - Membuat tabel.
• <tr> - Baris tabel.
• <th> - Header kolom.
• <td> - Data sel.

⚡ Pro Tips

Gunakan <th> untuk judul kolom agar data lebih terstruktur dan mudah dipahami.
