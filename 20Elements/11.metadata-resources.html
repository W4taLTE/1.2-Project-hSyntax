💡 Metadata Dasar

Metadata HTML membantu mesin pencari, browser, dan aplikasi memahami isi halaman.

metadata.html
<head>
  <title>My Website</title>
  <meta charset="UTF-8">
  <meta name="description" content="Deskripsi singkat halaman">
  <meta name="keywords" content="HTML, CSS, Web">
  <meta name="author" content="John Doe">
</head>

🔍 Penjelasan Detail:

• <title> - Judul halaman di tab browser
• description - Deskripsi singkat untuk SEO
• keywords - Kata kunci halaman
• author - Penulis halaman
• charset - Encoding karakter

⚡ Pro Tips

Gunakan deskripsi unik di setiap halaman agar SEO lebih optimal.
