üìù Basic Media Elements

üí° Dasar-dasar Media HTML

HTML menyediakan elemen khusus untuk menampilkan berbagai jenis media seperti gambar, audio, dan video.

media.html
<!-- Gambar dasar -->
<img src="image.jpg" alt="Deskripsi gambar" width="400">

<!-- Gambar responsif -->
<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 400px)" srcset="medium.jpg">
  <img src="small.jpg" alt="Gambar responsif">
</picture>

<!-- Audio dasar -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Browser Anda tidak mendukung elemen audio.
</audio>

üîç Penjelasan Detail:

‚Ä¢ <img> - Menampilkan gambar dengan atribut wajib src dan alt
‚Ä¢ <picture> - Menyediakan gambar responsif untuk berbagai ukuran layar
‚Ä¢ <audio> - Memutar file audio dengan kontrol pemutaran
‚Ä¢ controls - Menampilkan kontrol pemutaran default
‚Ä¢ <source> - Menentukan sumber media alternatif
‚Ä¢ srcset - Menyediakan berbagai versi gambar untuk perangkat berbeda
‚Ä¢ Selalu sertakan alt text untuk aksesibilitas

‚ö° Pro Tips

Gunakan atribut loading="lazy" pada gambar untuk optimasi performa (lazy loading)!

üîó Video & Embedded Content

üí° Video dan Konten External

HTML5 menyediakan elemen native untuk video dan embed konten dari platform lain.

video.html
<!-- Video HTML5 -->
<video width="640" height="360" controls poster="preview.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Browser Anda tidak mendukung elemen video.
</video>

<!-- YouTube Embed -->
<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>

<!-- Google Maps Embed -->
<iframe src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d..." width="600" height="450" style="border:0;" allowfullscreen></iframe>

üîç Penjelasan Detail:

‚Ä¢ <video> - Menampilkan video dengan berbagai format
‚Ä¢ poster - Gambar preview sebelum video diputar
‚Ä¢ autoplay - Memulai pemutaran otomatis (hati-hati penggunaannya)
‚Ä¢ loop - Video akan berulang setelah selesai
‚Ä¢ <iframe> - Menyematkan konten eksternal
‚Ä¢ YouTube dan Maps menyediakan kode embed yang bisa langsung digunakan
‚Ä¢ Atribut allowfullscreen memungkinkan mode layar penuh

‚ö° Pro Tips

Untuk video background, gunakan autoplay muted loop dan CSS position: fixed untuk efek yang menarik!

üé® Media Styling & Optimization

üí° CSS untuk Media Elements

Gunakan CSS untuk membuat elemen media lebih responsif dan menarik.

style.css
/* Gambar responsif */
img {
  max-width: 100%;
  height: auto;
  border-radius: 8px;
}

/* Video container responsif */
.video-container {
  position: relative;
  padding-bottom: 56.25%; /* 16:9 Aspect Ratio */
  height: 0;
  overflow: hidden;
}

.video-container iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}

/* Efek hover gambar */
img:hover {
  transform: scale(1.02);
  transition: transform 0.3s ease;
  box-shadow: 0 0 20px rgba(0, 188, 212, 0.5);
}

üîç Penjelasan Detail:

‚Ä¢ max-width: 100% - Membuat gambar scalable dengan container
‚Ä¢ height: auto - Mempertahankan aspect ratio gambar
‚Ä¢ Padding-bottom trick - Teknik untuk mempertahankan aspect ratio video
‚Ä¢ object-fit: cover - Berguna untuk gambar yang perlu mengisi container
‚Ä¢ Efek hover - Memberikan feedback visual saat interaksi
‚Ä¢ border-radius - Membuat sudut elemen media melengkung
‚Ä¢ Gunakan @media queries untuk penyesuaian di berbagai ukuran layar

‚ö° Pro Tips

Gunakan format gambar modern seperti WebP untuk kompresi yang lebih baik, dan sertakan fallback PNG/JPG!
