Panduan lengkap atribut global HTML yang dapat digunakan pada hampir semua elemen

📝Basic Tables

💡 Definisi

Atribut global adalah atribut yang dapat diterapkan pada hampir semua elemen HTML, memberikan fungsionalitas tambahan dan kontrol perilaku.

example.html
<p id="intro" class="highlight"> Hello World! </p>

🔍 Penjelasan Detail:

• id - Memberikan identitas unik pada elemen.
• class - Menentukan satu atau lebih kelas CSS untuk elemen.
• style - Menambahkan gaya inline langsung.
• title - Menyediakan teks tooltip saat kursor diarahkan.
• lang - Menentukan bahasa konten elemen.

⚡ Pro Tips

Gunakan id hanya sekali di satu halaman, dan manfaatkan class untuk gaya berulang.
